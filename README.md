AdventureWorks Sales Performance Dashboard

Multi-page interactive Power BI report for sales analytics and business intelligence


Overview
Built on the AdventureWorks enterprise dataset, this Power BI report transforms raw transactional data into an executive-ready, fully interactive dashboard. It covers global sales performance across countries, states, and cities — giving leadership a clear view of revenue, product portfolio, and customer base at every level.

Dual chart views (bar + pie) for proportional and absolute comparisons
